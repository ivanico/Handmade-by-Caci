### Task 1.3 â€” Auth Endpoints
Implement in `app/routers/auth.py`:

| Method | Path | Auth | Body | Returns |
|--------|------|------|------|---------|
| POST | `/api/auth/register` | none | `{email, password, full_name, phone?}` | `201 {id, email, full_name, role}` |
| POST | `/api/auth/login` | none | `{email, password}` | `200 {access_token}` + sets `refresh_token` httpOnly cookie |
| POST | `/api/auth/refresh` | refresh cookie | none | `200 {access_token}` + rotates cookie |
| POST | `/api/auth/logout` | none | none | `200` + clears cookie |
| GET | `/api/auth/me` | Bearer | none | `200 {id, email, full_name, role, phone}` |

- Duplicate email â†’ `400 {"detail": "Email already registered"}`
- Wrong password â†’ `401 {"detail": "Invalid credentials"}`
- **Done when**:
  ```bash
  curl -X POST http://localhost:8000/api/auth/register \
    -H "Content-Type: application/json" \
    -d '{"email":"test@test.com","password":"pass123","full_name":"Test User"}' # â†’ 201

  TOKEN=$(curl -s -X POST http://localhost:8000/api/auth/login \
    -H "Content-Type: application/json" \
    -d '{"email":"admin@shop.com","password":"admin123"}' | python3 -c "import sys,json; print(json.load(sys.stdin)['access_token'])")

  curl http://localhost:8000/api/auth/me -H "Authorization: Bearer $TOKEN" # â†’ 200 user object
  curl -X POST http://localhost:8000/api/auth/login -d '{"email":"admin@shop.com","password":"wrong"}' # â†’ 401
  ```
